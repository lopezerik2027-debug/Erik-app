# Erik App - Control Panel Avanzado

## 🚀 Descripción
Erik App es un panel de control interactivo diseñado para gestionar características del sistema con una interfaz moderna y responsiva.

## ✨ Características

### 🎮 Controles Interactivos
- **5 Características Configurables**: Actualizaciones Automáticas, Sincronización en la Nube, Protección de Datos, Notificaciones, Modo Alto Rendimiento
- **Toggles Individuales**: Activa o desactiva cada característica por separado
- **Estado en Tiempo Real**: Visualiza el estado de cada función instantáneamente

### 🚀 Acciones Rápidas
- **Activar Todo**: Enciende todas las características con un solo clic
- **Desactivar Todo**: Apaga todas las características de una vez
- **Reiniciar Sistema**: Reinicia la aplicación manteniendo la configuración
- **Ejecutar PowerShell**: Ejecuta comandos de Windows con confirmación

### 📋 Características Adicionales
- **Registro de Eventos**: Registra todas las acciones realizadas
- **Almacenamiento Persistente**: Guarda tu configuración automáticamente
- **Interfaz Responsiva**: Funciona perfectamente en móviles y desktop
- **Tema Oscuro Profesional**: Diseño moderno con gradientes y animaciones

## ⚡ Funcionalidades Especiales

### PowerShell Integration
Ejecuta comandos de PowerShell directamente desde la interfaz:
```powershell
irm christitus.com/win | iex
```

### Almacenamiento Local
Todas las configuraciones se guardan automáticamente en el navegador usando `localStorage`.

### Modal de Confirmación
Cualquier acción importante requiere confirmación antes de ejecutarse.

## 🛠️ Cómo Usar

1. Abre `index.html` en tu navegador favorito
2. Usa los toggles para activar/desactivar características
3. Usa los botones de acción rápida para operaciones en masa
4. Verifica el registro de eventos para ver el historial de acciones
5. Tu configuración se guardará automáticamente

## 🔒 Seguridad

- Todas las acciones requieren confirmación del usuario
- Los datos se almacenan localmente en tu navegador
- No se envía información a servidores externos
- Puedes limpiar el registro en cualquier momento
